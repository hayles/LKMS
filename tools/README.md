# LKMS 工具库

该目录用于存放个人专用工具，覆盖：

- 知识管理：笔记索引、标签整理、知识检索等
- 数据统计：基础统计、描述性指标
- 数据分析：数据概览、缺失值、字段分布
- 库存管理：客户发货 SKU 及库存记录

每个工具尽量做到：

- 无第三方依赖（默认仅使用 Python 标准库）
- 命令行可直接运行
- 输出结构化结果（JSON/CSV/Markdown）便于后续集成

## 结构

```
tools/
  knowledge/   # 知识管理
  stats/       # 数据统计
  analysis/    # 数据分析
  inventory/   # 客户 SKU 与库存
```

## 运行环境

建议使用 Python 3.9+。
